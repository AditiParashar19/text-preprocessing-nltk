# text-preprocessing-nltk
Python NLP text preprocessing using NLTK for tokenization, stopword removal, punctuation cleaning, and lemmatization.
# Text Preprocessing using NLTK (Python)

This project implements a Python-based text preprocessing pipeline using the NLTK library.

## Features
- Lowercasing
- Tokenization
- Stopword removal
- Punctuation filtering
- Lemmatization

## Requirements
- Python 3.x
- nltk
- string module
